JavaScript

Why is JavaScript popular in web development?

JavaScript is so popular in web development because it adds interesting effects and functionality to HTML, which is very limited as static markup language.

What are the great things about JavaScript?

Javascript can change HTML elements, attributes and styles. It can also include a lot of new features and functionality to the web page, since validate data to react to events or perform complex tasks.

What are its limitations?

The scope of javascript is more limited than the scope of the general programming language. One of the areas where this limitation is more remarkable is in the ability to operate with databases.
